# FoodX

Food delivery web application using the Razorpay API

Features:

* Add to cart functionality
* Remove from cart functionality
* Product Filtering functionality
* Cart container
* Modern UI design
* Login to google account
* Payments like UPI,card.

## Instructions

### `yarn start`
